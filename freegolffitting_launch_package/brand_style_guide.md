# Brand Style Guide (FreeGolfFitting by GolfMax)

## Voice & Messaging
- **Value prop:** Remote Golf Club Fitting — pro-grade specs from real fitters.
- **Tone:** Direct, confident, helpful. No jargon unless necessary.
- **Tagline (optional):** “Get fit right, from anywhere.”

## Logo
- Wordmark: “FreeGolfFitting” + small “by GolfMax” lockup.
- Safe area: 1× cap height. Minimum size: 24px height on web.
- Do not skew, stroke, or place on busy backgrounds.

## Colors
- **Primary:** Mint Green `#4ADE80` (CTA gradient start)
- **Secondary:** Cyan `#22D3EE` (CTA gradient end)
- **Text:** Off‑white `#F6F7F8` on dark
- **Surface:** `#111315` / Alt `#0F1216`
- **Borders:** `#23272E`
- **Background:** Near‑black `#0A0A0A`

## Typography
- **Headlines:** DM Serif Display
- **Body/UI:** Inter
- Sizes: H1 32–52px, H2 24–32px, body 16–18px, small 14px.

## Spacing & Layout
- Grid: 12‑column, max width 1100px, 16px base spacing.
- Corners: 16px; Shadows: soft (10/30 rgba 0,0,0,.35).

## Components
### Buttons
- **Primary:** Rounded pill, gradient mint→cyan, dark text.
- **Secondary:** Transparent with border, white text.
- Hover: raise by 1px, slight opacity.

### Cards
- Surface‑alt, 1px border, 16px radius, generous padding.

### Forms
- Clear labels, inline validation, helpful placeholder text.
- Privacy note beneath primary CTA.

## Imagery
- Clean, modern golf visuals. Favor motion blur / trajectory overlays.
- Expose equipment detail shots in steps/benefits.
